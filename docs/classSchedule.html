◆ crossover()

Schedule * Schedule::crossover ( Schedule &  parent2) const
inline

Crosses over this schedule with another scheduling and returns a pointer to the new schedule. During the crossover this schedule is copied and then a number of points defined in m_numberOfCrossoverPoints is taken from the other course given as parameter to this function.

◆ mutation()

void Schedule::mutation ( )
inline

Mutates this schedule in place by changing a random selection of classes to random classrooms and time slots. Number of points mutated is defined in the m_mutationSize variable. If variable mutations is enabled, then a random value between 0 and m_mutationSize is used instead.
