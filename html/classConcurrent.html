Detailed Description

A class that holds all the information regarding course concurrency. Two courses are considered concurrent if they have been taken simultaniously in past terms. And the number of students that took them together is the concurrency value of these two courses.

◆ m_cuda_matrix

int* Concurrent::m_cuda_matrix
private

An alternative to the m_map member variable. This pointer holds a 2D matrix that is implemented in 1D. So given two courses, the concurrency value of them is in m_cuda_matrix[courseId1 * size + courseId2] (order does not matter). This is used specifically for CUDA.
