Detailed Description

A class representing 1 hour of class. It is basicly a three-way association between a course, a classroom and a timeslot.

◆ m_root

bool CourseTime::m_root
private

This flag is set to true only for the very first hour of a consecutive set of course times. For example if the course CS406 has a 2 hour class on a day, then only the first hour has m_root set to true. This is used to preserve this structure of consecutive classes during mutations and crossovers.
